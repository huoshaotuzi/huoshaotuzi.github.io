使用 cpolar 实现内网穿透（免费版）
2023-04-23|黑科技

前言

在学习内网穿透之前需要先了解什么是内网，什么是外网。

大学的时候我跟舍友一起玩 War3，可以利用本地局域网实现联机对战，而最简单的方式就是直接用一条网线把两台电脑连起来，这就组成了局部网络连接，简称局域网，局域网就是网线连接起来那几台电脑组成的网络，只有连着线的电脑可以互相访问，没用网线连接的就访问不了。

将局域网的范围扩大，比如用网线连接隔壁宿舍的电脑，隔壁的隔壁……一直连线，直到整个学校所有电脑都连在一起，那就叫做“校园网”，即整个学校区域的电脑都可以互相访问，再扩大范围，把所有学校的电脑都连接在一起，甚至把普通家庭、公司企业……所有的电脑都连在一起，那就叫做「万维网」。

为了区别不同的电脑，有一个概念叫做「IP地址」，每家每户的电脑都分配一个 IP 地址，根据 IP 地址就能找到对应的电脑，这个 IP 地址就叫做「公网 IP」，公网 IP 是万维网中任何人都可以访问的 IP 地址，但是某些情况下我们并不希望自己的 IP 地址让别人访问，但是我们需要像校园网那种小范围可以互相访问的网络，这个就叫做内网 IP，内网 IP 用于区分局部的主机，局域网内通过内网 IP 可以实现互相访问。

如果你家里的电脑想让其他人访问到，那就必须要有一个「公网 IP」，但是通常情况下我们都只有「内网 IP」，这样你电脑上的东西别人是访问不到的，因此我们大多数情况下都会去买云服务器，然后把东西部署在云服务器上面，腾讯或者阿里这些大厂可以拿到很多公网 IP，我们只要买他们的服务器就可以免费分配一个公网 IP，通过公网 IP 全世界都可以访问到我们的服务器。

现在，我们如果想要让家庭电脑变成一台服务器，让全世界都可以访问呢？
我们是没有公网 IP 的，内网 IP 别人也访问不到，因此就需要用一种特殊手段「内网穿透」。

内网穿透

正如字面意思，外网跟内网之间如同有一层保护膜，通过特殊手段穿透这层保护膜，使得外网也能进入到内网，这种手段就是通过端口映射，然后找一个中间的代理负责转发，换句话说就是要借助一个中间商的力量，我们只需要配置好要暴露的端口，在本地跑一个程序，让中间商可以获取到我们暴露的端口，然后中间商再分配一个公网可以访问的域名，外部就可以通过这个域名访问到我们本地暴露的端口了。

比如有个用户小明现在访问了这个中间商分配的域名，中间商再去读取你本地暴露的端口，将请求转发给小明，为什么中间商可以“穿透”内网访问到我们本地暴露的端口？因为我们本地要跑一个程序，是你主动将端口暴露给中间商的，程序关掉的话，中间商也获取不到了。

中间商的作用就是“牵线搭桥”的。
